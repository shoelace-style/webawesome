# Shadow Tokens

Shadow tokens are used to give elements the appearance of being raised off the page. Use them with the `box-shadow` property. These are especially useful for menus, popovers, and dialogs.

| Token                        | Example                                                                              |
| ---------------------------- | ------------------------------------------------------------------------------------ |
| `--wa-shadow-resting`        | <div class="shadow-demo" style="box-shadow: var(--wa-shadow-resting);"></div>        |
| `--wa-shadow-raised`         | <div class="shadow-demo" style="box-shadow: var(--wa-shadow-raised);"></div>         |
| `--wa-shadow-raised-higher`  | <div class="shadow-demo" style="box-shadow: var(--wa-shadow-raised-higher);"></div>  |
| `--wa-shadow-raised-highest` | <div class="shadow-demo" style="box-shadow: var(--wa-shadow-raised-highest);"></div> |
